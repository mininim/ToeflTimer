## 1️⃣ Service Name
### Toefl Timer  
## 2️⃣ Description
***Get a high score in the TOEFL Speaking section with Toefl Timer!***  

□ Services

1. **Timer**  
Through UI based on the actual test, you can check the preparation time and speaking time for each task. 
2. **Delay**  
It is designed to take into account the delay according to guidance such as “Prepare your response after the Beep”. You can practice in an environment that is very similar to the actual test.
